###Flexible Media:
<p>Images, videos and media need to be scalable as per the viewports.
To make the media fit to the viewport, one way is to set the <code>max-width</code>
property to 100%.</p>

<pre><code>img, video, canvas {
  max-width: 100%;
}
</code></pre>
